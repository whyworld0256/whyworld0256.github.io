离散化(要求保序)

离散化是在不改变数据相对大小的条件下，对数据进行相应的缩小。

此时我们有$10^5$ 个数，但是数的取值范围为$10^9$, 如果在某个问题中我们需要将这些数作为下标使用，那么我们需要去开一个长度为$10^9$ 的数组，但是我们只会用到其中$10^5$ 个位置，因此这样会造成很大的空间浪费。此时我们引出离散化。

如果我们不需要这些数的具体数值，那么我们完全可以将这$10^5$ 个取值范围为$10^9$ 的数映射到 $0\sim 10^5$ ，这样我们只需要一个长度为$10^5$ 的数组即可。

如何映射呢：现在给我们5个数：$1,5,3,6,10000$

我们先将数组排序，然后把每个数映射到它的下标即可了,并且在这之前我们需要去重, 否则同样的数却映射到不同的值去了。

例题

假定有一个无限长的数轴，数轴上每个坐标上的数都是 0。

现在，我们首先进行 n 次操作，每次操作将某一位置 x 上的数加 c。

接下来，进行 m 次询问，每个询问包含两个整数 l 和 r，你需要求出在区间 [l,r] 之间的所有数的和。

输入格式

第一行包含两个整数 n 和 m。

接下来 n 行，每行包含两个整数 x 和 c。

再接下来 m 行，每行包含两个整数 l 和 r。

输出格式

共 m 行，每行输出一个询问中所求的区间内数字和。

数据范围

$−10^9≤x≤10^9,$
$1≤n,m≤10^5,$
$−10^9≤l≤r≤10^9,$
$−10000≤c≤10000$
