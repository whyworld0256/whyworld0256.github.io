二叉树的最大深度

题目描述

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

解答

递归求解，一棵树的最大深度是左右子树最大深度的最大值并加1。 这样我们就找到了完全相同的子问题，因此用递归求解。

相同的树

题目描述

给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

解答

两棵树相同等价于当前节点相同，然后对应左子树相同，对应右子树相同。

递归终止条件，p 和 q 中有一个为空。

平衡二叉树

题目描述

给定一个二叉树，判断它是否是高度平衡的二叉树。

本题中，一棵高度平衡二叉树定义为：

一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1 。

题解

判断一棵树为平衡二叉树，我们需要判断其左右子树的高度差的绝对值小于等于1，并且左右子树都必须是平衡二叉树

叶子节点 是指没有子节点的节点。

路径总和 III

题目描述

给定一个二叉树的根节点 root ，和一个整数 targetSum ，求该二叉树里节点值之和等于 targetSum 的 路径 的数目。

路径 不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。

解答

此题比较难，我们考虑使用前缀和来计算每条路径的和。

每个节点的前缀和表示从根节点到它这条路径的和。

那么两个节点的前缀和之差就是路径和(不包含父节点)

二叉树的前序遍历

题目描述

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

解答

什么是前序遍历。对每一棵子树，先输出根节点，再输出左节点，最后输出右节点。

递归公式：preorder(node) = print node —> preorder(node->left) --> preorder(node->right)

二叉树的后序遍历

题目描述

给你一棵二叉树的根节点 root ，返回其节点值的 后序遍历

解答

二叉树的后序遍历的记忆法则是“左右根”，即先遍历左子树节点，再遍历右子树节点，最后遍历根节点。

递归公式为：preorder(node) = preorder(node->left) --> preorder(node->right) —> print node

二叉树的中序遍历

题目描述

给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。

解答

二叉树的中序遍历的记忆法则是“左根右”，即先遍历左子树节点，再遍历根节点，再遍历右子树节点。

preorder(node) = preorder(node->left) --> print node —> preorder(node->right)
