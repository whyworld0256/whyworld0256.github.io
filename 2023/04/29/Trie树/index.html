查找

模板

维护一个字符串集合，支持两种操作：

  1. I x 向集合中插入一个字符串 x

  2. Q x 询问一个字符串在集合中出现了多少次。

共有 N个操作，所有输入的字符串总长度不超过 10^5^，字符串仅包含小写英文字母。

解答

练习： 最大异或对

题目描述

在给定的 $N$个整数$A_1 \cdots A_n$ 中选出两个进行 异或运算，得到的结果最大是多少？

数据范围

$1\le N \le 10^5$,
$0\le A_i <2^{31}$

解答

我们给定 $A_i$ ，现在去寻找$A_1 \dots A_i-1$ 中与 $A_i$ 异或最大的数。

把$A_i$ 转化为二进制，从高位向低位去对比，每次都尽量去找某些数，使得二者对应的那位数是不同的。如果没有，那就只能要相同的。

每个点的子节点对多只有两个，因此我们从根节点开始，每次都去走与$A_i$ 不同的那个分支，如果只有一种选择，那就直接走。最后走到终点。
