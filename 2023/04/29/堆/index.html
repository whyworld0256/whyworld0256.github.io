堆

基本操作

  • 插入一个数
  • 求集合中的最小值

  • 删除最小值

  • 删除任意一个元素

  • 修改任意一个元素

基本结构

堆是一棵完全二叉树，即除了最后一层，都是满的，并且最后一层从左往右排列。

小根堆：

递归定义：根节点的值小于等于左右子节点的值。

因此根节点是所有数据的最小值。

存储

一维数组去存，数组下标从1开始，$x$ 的左儿子是 $2x$, 右儿子是$2x+1$

两个基本操作

首先，这两个操作都是在改变了某个元素的值后，我们应该怎么去调整这棵树使其成为一个新的小根堆。

  • down: 把某个数调小，它有可能就要往下去寻找属于它的新的位置
  • up: 把某个数调大，它就往上浮。

此时，我们可以由这两个基本操作得到所有的基本操作。

基本操作的实现

  • 插入一个数：在数组最后加入这个数，然后up
  • 求集合中的最小值: 访问堆定元素即可
  • 删除最小值：将堆顶元素赋值为数组末尾元素，删除数组末尾元素，然后down
  • 删除任意一个元素：将那个元素赋值为数组末尾元素，此时这个数有可能往上走，也有可能往下走，但二者只能选其1。
  • 修改任意一个元素： 先赋值，然后和上面情况相同。

1.堆排序

输入一个长度为 n 的整数数列，从小到大输出前 m小的数。

输入格式

第一行包含整数 n和 m。

第二行包含 n 个整数，表示整数数列。

输出格式

共一行，包含 m 个整数，表示整数数列中前 m 小的数。

2. 模拟堆

题目描述

维护一个集合，初始时集合为空，支持如下几种操作：

  1. I x，插入一个数 x；

  2. PM，输出当前集合中的最小值；

  3. DM，删除当前集合中的最小值（数据保证此时的最小值唯一）；

  4. D k，删除第 k个插入的数；

  5. C k x，修改第 k个插入的数，将其变为 x；

现在要进行 N次操作，对于所有第 2 个操作，输出当前集合的最小值。

输入格式

第一行包含整数 N。

接下来 N 行，每行包含一个操作指令，操作指令为 I x，PM，DM，D k 或 C k x 中的一种。

输出格式

对于每个输出指令 PM，输出一个结果，表示当前集合中的最小值。每个结果占一行。
