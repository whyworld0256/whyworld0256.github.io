滑动子数组的美丽值

题目描述

给你一个长度为 n 的整数数组 nums ，请你求出每个长度为 k 的子数组的 美丽值 。

一个子数组的 美丽值 定义为：如果子数组中第 x 小整数 是 负数 ，那么美丽值为第 x 小的数，否则美丽值为 0 。

请你返回一个包含 n - k + 1 个整数的数组，依次 表示数组中从第一个下标开始，每个长度为 k 的子数组的 美丽值 。

  • 子数组指的是数组中一段连续 非空 的元素序列。

提示：

  • n == nums.length
  • 1 <= n <= 105
  • 1 <= k <= n
  • 1 <= x <= k
  • -50 <= nums[i] <= 50

题解

问题的关键在于如何快速地求出每个滑动窗口中地第x小的数。

如果你知道桶排序的话，可以产生以下思路。

我们维护一个$cnt$ 数组，其长度为$101$ 正好使$nums[i]$ 的范围，$cnt[i]$ 表示在当前这个滑动窗口中，$i$ 出现的次数。

若$s$ 是第$x$小的数，表明比它小的数的个数$<x$ ， 大于等于它的数的个数$\ge x$ ，因此在当前这个滑动窗口下，我们从$-50$ 遍历到$0$

不断累加$cnt[i]$, 如果此时累加值刚好大于等于$x$ 那么表明小于等于$i$ 的数的个数刚好大于等于$x$ 那么$i$ 便是答案。如果加到$cnt[0]$ 后还没有超过$x$ 那么表明第$x$小的数一定大于0，因此$ans[i]=0$ 。

时间复杂度: $O(50n)$
